###Week 1: Writing code/ setting your working directory

* ```R
  myfunction <- function(x){
      y <- rnorm(100)
      mean(y)
  }
  
  second <- function(x){
      x + norm(length(x))
  }
  ```

* save functions as function.R | `source("myfunction.R")` | `myfunction(x) or second(x)`
